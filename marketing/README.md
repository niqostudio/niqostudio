# marketing

NIQO STUDIO の発信資産（声・世界観・戦略・投稿済み履歴）。X・note の投稿下書きを、これらの資産から生成するための土台。

## 構成

| パス | 役割 |
| --- | --- |
| [brand/voice.md](brand/voice.md) | 声・トーン・do/don't |
| [brand/worldview.md](brand/worldview.md) | 発信の旗となる世界観（テーゼ） |
| [brand/audience.md](brand/audience.md) | ターゲットペルソナ |
| [strategy/posting-strategy.md](strategy/posting-strategy.md) | 柱・チャネル分担・カデンス・観測シグナル・アンチパターン |
| [history/x.md](history/x.md) / [history/note.md](history/note.md) | 投稿済み（公開済み）本文の記録 |

## 使い方

生成コマンドは各プロダクトのリポジトリ（非公開）側に置く。コマンドはこのモジュールの資産を読み、プロダクト固有素材（非公開）と合わせて下書きを生成し、下書きは非公開側に出力する。投稿後、本文を `history/` に記録する。

このリポジトリは public のため、**未公開プロダクトの名前・固有機構名はここに書かない**（[CLAUDE.md](CLAUDE.md)）。
