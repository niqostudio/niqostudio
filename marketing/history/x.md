# history — X 投稿記録

X に投稿した（＝公開済みの）本文を手動で記録する。生成時に重複回避と「どの切り口が刺さったか」の学習材料になる。

柱は [posting-strategy.md](../strategy/posting-strategy.md) 参照（柱1 理解負債／柱2 受託のリアル／柱3 思想／柱4 build-in-public）。自己紹介・お知らせ等は「その他」でよい。

記録は新しいものを上に追記する。形式：

```
## YYYY-MM-DD / 柱: <柱名 or その他> / 形式: 単発 | スレッド
<投稿した本文（スレッドは各ポストを --- で区切る）>

反応メモ: <任意。誰が・どんな反応か。空でも可>
```

未投稿の下書きはここに置かない（非公開側の `marketing/drafts/`）。仮名・固有機構名を含む本文はここに記録しない（層1の投稿は元々それらを含まない想定）。
